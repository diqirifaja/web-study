Pengertian HTML

HTML adalah bahasa markup yang digunakan untuk membuat struktur dasar dan tampilan situs web. HTML digunakan untuk membuat elemen-elemen seperti teks, gambar, dan hyperlink. Dokumen HTML disajikan pada web browser.
